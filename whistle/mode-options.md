# 启动参数 `-M m1|m2|mX` 的用途

可以通过启动参数 `-M m1|m2|mX` 设置或禁用 Whistle 的一些默认行为，其中 `m1`、`m2`、... 可以为以下参数的任意组合：

### disableForwardedProto

### disableForwardedHost

### network

### rules

### rulesOnly

### plugins

### pluginsOnly

### shadowRules

### shadowRulesOnly

### enableMultipleRules

### notAllowDisableRules
