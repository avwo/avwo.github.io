rule

响应规则列表

  1. 请求替换
  2. file (替换本地文件)
  3. xfile (替换本地文件，如果本地文件不存在，则请求线上)
  4. rawfile (替换本地http响应内容格式的文件)
  5. xrawfile (替换本地http响应内容格式的文件，如果本地文件不存在，则请求线上)
  6. tpl (替换本地目标文件，可用于模拟jsonp请求)
  7. xtpl (同上，与xfile类似)
  8. proxy (代理到其它http代理服务器)
  9. socks (代理到其它socks代理服务器)
  10. 自定义
