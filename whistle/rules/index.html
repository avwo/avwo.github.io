协议列表

协议列表

  1. host (设置host)

  2. rule (设置响应规则)

    • 请求替换
    • file (替换本地文件)
    • xfile (替换本地文件，如果本地文件不存在，则请求线上)
    • rawfile (替换本地http响应内容格式的文件)
    • xrawfile (替换本地http响应内容格式的文件，如果本地文件不存在，则请求线上)
    • tpl (替换本地目标文件，可用于模拟jsonp请求)
    • xtpl (替换本地目标文件，可用于模拟jsonp请求，则请求线上)
    • proxy (代理到其它http代理服务器)
    • socks (代理到其它socks代理服务器)
    • 自定义

  3. weinre (设置weinre，调试手机页面)

  4. log (打印网页js错误或者调试信息)
  5. pac (设置pac脚本)
  6. filter (过滤规则，隐藏请求等)
  7. disable (禁用缓存、cookie等)
  8. delete (删除指定的字段)
  9. plugin (通过插件获取请求状态及设置新规则)
  10. dispatch (动态修改请求url的参数)
  11. urlParams (修改请求url的参数)
  12. urlReplace (通过正则或字符串替换请求url，类似str.replace)
  13. method (修改请求方法)
  14. statusCode (直接响应)
  15. replaceStatus (替换后台的响应状态码)
  16. hostname (修改请求头部的host字段)
  17. referer (修改请求referer)
  18. accept (修改请求头的accept)
  19. auth (修改请求用户名密码)
  20. etag (修改请求头部的etag)
  21. ua (修改请求user-agent)
  22. cache (修改缓存策略)
  23. redirect (302重定向)
  24. location (设置响应头部的location字段)
  25. attachment (设置下载头部)
  26. params (修改请求参数)
  27. html (往响应为html的内容后面追加数据)
  28. css (往响应为html或css的内容后面追加数据)
  29. js (往响应为html或js的内容后面追加数据)
  30. req (修改请求属性)
  31. res (修改响应属性)
  32. reqDelay (延迟请求)
  33. resDelay (延迟响应)
  34. reqSpeed (限制请求速度)
  35. resSpeed (限制响应速度)
  36. reqHeaders (修改请求头)
  37. resHeaders (修改响应头)
  38. reqType (修改请求类型)
  39. resType (修改响应类型)
  40. reqCharset (修改请求的编码)
  41. resCharset (修改响应的编码)
  42. reqCookies (修改请求cookies)
  43. resCookies (修改响应cookies)
  44. reqCors (修改请求cors)
  45. resCors (修改响应cors)
  46. reqPrepend (往请求内容前面添加数据)
  47. resPrepend (往响应内容前面添加数据)
  48. reqBody (替换请求内容)
  49. resBody (替换响应内容)
  50. reqAppend (往请求内容后面追加数据)
  51. resAppend (往响应内容后面追加数据)
  52. reqReplace (通过正则或字符串替换请求文本内容，类似str.replace)
  53. resReplace (通过正则或字符串替换响应文本内容，类似str.replace)
  54. reqWrite (将请求内容写入指定的文件)
  55. resWrite (将响应内容写入指定的文件)
  56. reqWriteRaw (将请求的完整内容写入指定的文件)
  57. resWriteRaw (将响应的完整内容写入指定的文件)
  58. exportsUrl (把请求的url列表按顺序导出到指定文件)
  59. exports (导出请求数据到指定文件)
